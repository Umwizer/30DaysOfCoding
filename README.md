# 30DaysOfCoding
# becoming-a-better-engineer

I'll be honest. I've been relying on AI too much, not just for hints, for *everything*. Open a blank file, freeze, paste into ChatGPT, and copy the answer. And at the end, I've "completed" something I don't fully understand.

That stops here. 30 days of me actually coding.

## Mission

- Solve problems independently without freezing at a blank file
- Write code from scratch, even when it's messy the first time
- Think like a backend engineer
- Develop a testing mindset
- Document the journey publicly

## The 70/30 Rule

**70% me. 30% AI.**

AI can review, explain, hint, or help debug *after* I've tried. AI cannot write the initial solution. If I open a blank file and immediately ask AI to fill it — I've failed the challenge.

If I miss a day, the next day I do double.
